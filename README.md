# Full-Stack POS Billing Terminal

An automated Point of Sale (POS) application featuring a real-time React front-end dashboard and a Yii2 RESTful API backend architecture. Built for seamless retail and wholesale transaction logging with integrated hardware barcode scanning support.

## 🚀 Features
* **Dynamic Cart Operations:** Live tax calculations (GST / CGST + SGST mapping).
* **Sequential Invoicing:** Automated chronological invoice sequence generation fetched from backend API endpoints.
* **Hardware Integration:** Global key-listener tracking for seamless plug-and-play USB barcode scanners.
* **Thermal Printing:** Native 2-inch continuous thermal receipt strip print-overrides formatted using pure CSS.
